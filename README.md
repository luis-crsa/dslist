# Tecnologias utilizadas
- Java
- Spring Boot
- JPA / Hibernate
- Maven

# Como executar o projeto
Pré-requisitos: Java 21

```bash
# Clonar repositório
git clone https://github.com/luis-crsa/dslist.git

# Acessar a pasta do projeto
cd dslist

# Executar o projeto
./mvnw spring-boot:run
```
